Brevix Settings

Browser extension v0.1.0 — runs on claude.ai and chatgpt.com.

Default mode

Applied to every chat conversation until you change it.

Behavior

When off, the rule prefix is added to every message — wastes tokens but is more reliable.

Removes the visible Brevix rule text from your own message bubbles. Rule still rides in the API request.

About

Brevix is a deterministic rule engine for compressing LLM output. The browser extension nudges Claude/ChatGPT to apply the same rules locally inside the web chat, since these tools don't expose API access.

Source: github.com/Yash-Koladiya30/brevix
